# Web3 Hackathon Starter Template

A beginner-friendly React + TypeScript starter template for the Web3 hackathon.

## Prerequisites

- [GitHub account](https://github.com)
- [Git](https://git-scm.com/downloads)
- [Node.js](https://nodejs.org/) (v18 or higher)

## Getting Started

1. **Fork this repo** (click Fork at the top right)
2. **Clone your fork**
   ```bash
   git clone https://github.com/YOUR_USERNAME/web3-starter-template
   cd web3-starter-template
   ```
3. **Install dependencies**
   ```bash
   npm install
   ```
4. **Start the dev server**
   ```bash
   npm run dev
   ```
5. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Project Structure

- `src/components/` — Reusable UI components
- `src/pages/` — Route-level pages
- `src/App.tsx` — App entry + routing
